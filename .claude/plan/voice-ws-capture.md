# Plan: Voice Capture via WebSocket (Phase 1)

## Scope

Minimal working demo: browser button → mic capture → WebSocket → Node backend.
No AI, no TTS, no sales logic yet.

## Tech Stack

- **Client**: React 18 + Vite + TypeScript (fresh scaffold)
- **Server**: Node.js + Express + `ws` library (fresh scaffold)
- **Audio**: `MediaRecorder` API → chunks as `ArrayBuffer` → WS binary frames
- **Workspace**: npm workspaces (root `package.json` ties both together)

## Client: VoiceButton UX

- States: `idle` | `connecting` | `recording` | `error`
- **Click "Start"** → request mic permission → open WS → send `{type:"session.start"}` → start MediaRecorder → stream chunks
- **Click "Stop"** → stop MediaRecorder → send `{type:"session.end"}` → close WS → return to idle
- Button label reflects state; show chunk counter to confirm data is flowing

## Client: useVoiceCapture Hook

```
function useVoiceCapture(wsUrl: string) {
  // refs: ws, mediaRecorder, stream
  // state: status, chunkCount

  start():
    1. navigator.mediaDevices.getUserMedia({ audio: true })
    2. new WebSocket(wsUrl)
    3. ws.onopen → send JSON {type:"session.start"}
    4. new MediaRecorder(stream, { mimeType: "audio/webm;codecs=opus" })
    5. recorder.ondataavailable → blob.arrayBuffer() → ws.send(buffer)
    6. recorder.start(250)   // 250ms chunks

  stop():
    1. recorder.stop()
    2. ws.send(JSON {type:"session.end"})
    3. ws.close()
    4. stream.getTracks().forEach(t => t.stop())
}
```

## Server: index.js

```javascript
const express = require("express");
const { WebSocketServer } = require("ws");
const http = require("http");

const app = express();
const server = http.createServer(app);
const wss = new WebSocketServer({ server, path: "/ws" });

wss.on("connection", (ws) => {
  let chunkCount = 0;

  ws.on("message", (data, isBinary) => {
    if (!isBinary) {
      const msg = JSON.parse(data.toString());
      console.log("[WS] control:", msg.type);
    } else {
      chunkCount++;
      console.log(`[WS] audio chunk #${chunkCount}: ${data.length} bytes`);
      // Future: pipe to STT service
    }
  });

  ws.on("close", () => {
    console.log(`[WS] session ended. Total chunks: ${chunkCount}`);
  });
});

server.listen(8080, () => console.log("API listening on :8080"));
```

## Risks

| Risk                                          | Mitigation                                             |
| --------------------------------------------- | ------------------------------------------------------ |
| Mic permission denied                         | Show clear error state in UI                           |
| `audio/webm;codecs=opus` unsupported (Safari) | Fallback to `audio/webm` or `audio/mp4`                |
| WS proxy not working in Vite                  | Use full URL `ws://localhost:8080/ws` directly for dev |
| MediaRecorder chunks arrive after `stop()`    | Flush remaining chunk before closing WS                |
