## ✨ Features

- ⚡ **Bun-powered** toolchain for fast installs and runs
- 🧩 **Split client / API** — clean frontend/backend separation
- 📦 **Product & listing** management
- 🔍 Built-in research notes (`deep-research-report.md`) documenting the approach

## 🚀 Quick start

```bash
bun install
bun run dev        # client + api (see package.json scripts)
```
